Для запуска необходимо определить переменный config.env:

- NODE_ENV
- PORT
- DATABASE
- DATABASE_PASSWORD
- JWT_SECRET
- JWT_EXPIRES_IN
- JWT_COOKIE_EXPIRES_IN
- EMAIL_USERNAME
- EMAIL_PASSWORD
- EMAIL_HOST
- EMAIL_PORT

Команды для запуска:
- `npm run start:dev` - Дев сборка 
- `npm run start:prod` - Прод сборка
- `npm run debug` - Запуск отладчика
